Minimum background time (in minutes) Dustman only closes tabs which have been in the background for this long.
Minimum number of tabs per window Dustman closes tabs only in windows that have more than this many tabs. Pinned tabs are not counted towards this limit, and never auto-closed. While unpinned tabs that play sound are counted towards this limit, they are also never auto-closed.
Number of saved tabs Dustman can save a number of tabs it has closed. The list can be accessed via the garbage bin button in the toolbar. Set to 0 to disable saving tabs altogether.
Clear saved tabs when Firefox closes
Don't auto-close tabs in groups When enabled, tabs that belong to a tab group will not be automatically closed.
